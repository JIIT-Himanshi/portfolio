# Himanshi Chauhan | Portfolio

A modern and responsive personal portfolio website built to showcase my projects, technical skills, and design approach as a Frontend Developer.

## Highlights

- Cinematic hero section with smooth motion effects
- Clean and minimal UI with glassmorphism styling
- Responsive design for desktop, tablet, and mobile devices
- Smooth scrolling animations using Framer Motion and GSAP
- Interactive custom cursor effect
- Light and dark themed interface

## Tech Stack

React.js • Vite • Tailwind CSS • Framer Motion • GSAP • JavaScript • HTML5 • CSS3

## Getting Started

### Prerequisites

- Node.js (v18 or above)
- npm

### Installation

```bash
git clone https://github.com/JIIT-Himanshi/portfolio.git
cd portfolio
npm install
npm run dev
```

## Available Scripts

```bash
npm run dev
```
Starts the development server.

```bash
npm run build
```
Creates an optimized production build.

```bash
npm run preview
```
Previews the production build locally.
